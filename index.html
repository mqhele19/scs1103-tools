🤔 How many things were involved?

Your wireless mouse just moved the cursor.
How many distinct steps do you think happened?

?

✅ 7 distinct steps!

  • Mouse hardware (sensor)
  • Radio signal (Bluetooth / 2.4GHz)
  • Bluetooth chip (hardware)
  • Interrupt (CPU stops)
  • CPU processes data
  • Bluetooth driver (translates)
  • OS moves the cursor

Each step involves hardware, firmware, drivers, the OS, interrupts, and abstraction. Let's see how it works.

📡 USB Dongle vs Bluetooth

Two different paths to the same place — see where they diverge and converge.

🔌 USB Dongle

2.4GHz Radio
  • Mouse sends radio signal
  • Dongle receives it
  • USB port gets data
  • USB driver reads it
  • HID driver interprets
  • OS moves cursor

📶 Bluetooth

Bluetooth Radio
  • Mouse sends radio signal
  • Bluetooth chip receives it
  • Bluetooth stack processes
  • Bluetooth driver reads it
  • HID driver interprets
  • OS moves cursor
⬇️ Both converge at the HID driver — that's abstraction!

🎭 Virtual Device Abstraction

Same top layers every time. Only the bottom layer swaps. Watch what changes — and what doesn't.

🎭 The Application, OS, and Driver layers never changed. Only the bottom layer did — that's the power of abstraction.
